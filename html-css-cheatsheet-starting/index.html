CSS Layout Properties Reference

Property Name Description
float Float The 'float' property is used for positioning elements in a document flow. It allows an element to be taken out of the normal flow and moved to the left or right side of its containing element.
display: flex; Flexbox Flexbox is a one-dimensional layout model that allows elements within a container to be dynamically arranged. It provides powerful tools for distributing space, aligning items, and creating responsive layouts.
display: grid; Grid Grid is a two-dimensional layout model that allows you to create complex layouts by defining rows and columns. It offers precise control over item placement and alignment.
Property Name Description
text-transform Text Transform The 'text-transform' property is used to control the capitalization of text. It can be used to make text uppercase, lowercase, capitalize the first letter of each word, or capitalize all words.
box-shadow Box Shadow The 'box-shadow' property adds a shadow effect to an element. It allows you to control the shadow's position, size, blur, and color, which can create depth and visual interest in your design.
transition Transition The 'transition' property is used to add smooth transitions between different states of an element. It allows you to specify the properties to transition, duration, timing function, and delay.
